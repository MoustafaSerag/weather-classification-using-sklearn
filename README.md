# 🌦️ Weather Image Classification using Scikit-Learn

A machine learning project that classifies weather conditions from images using traditional computer vision techniques and Scikit-Learn.

## 📌 Features

- Weather image classification
- Feature extraction from images
- Scikit-Learn classifier
- Model serialization using Joblib/Pickle
- Easy prediction pipeline

## 🛠️ Technologies

- Python
- Scikit-Learn
- OpenCV
- NumPy
- Pillow

## 📂 Project Structure

```
weather-classification-using-sklearn/
│
├── main.py
├── requirements.txt
├── model.p
├── data.txt
└── .gitignore
```

## 🚀 Installation

```bash
git clone https://github.com/MoustafaSerag/weather-classification-using-sklearn.git

cd weather-classification-using-sklearn

pip install -r requirements.txt
```

## ▶️ Run

```bash
python main.py
```

## 📊 Dataset

The dataset is not included in this repository due to its size.

Download the dataset and place it in the project directory before running the project.

## 📈 Model

The trained model is stored in:

model.p
